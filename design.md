# Design Style Guide - Vehicle Insurance Broker Website

## Design Philosophy

### Color Palette
- **Primary**: Deep Navy (#1a365d) - Trust, stability, professionalism
- **Secondary**: Warm Orange (#ed8936) - Energy, urgency, call-to-action
- **Accent**: Light Gray (#f7fafc) - Clean, modern background
- **Text**: Charcoal (#2d3748) - High contrast, readable
- **Success**: Green (#38a169) - Positive actions, confirmations
- **Warning**: Amber (#d69e2e) - Alerts, important information

### Typography
- **Display Font**: Tiempos Headline - Bold, authoritative for headings
- **Body Font**: Suisse Int'l - Clean, readable for content
- **UI Font**: Inter - Modern, technical for forms and interfaces

### Visual Language
- Professional and trustworthy aesthetic
- Clean, uncluttered layouts with generous white space
- Subtle shadows and rounded corners for modern feel
- High contrast for accessibility
- Consistent iconography throughout

## Visual Effects & Animations

### Used Libraries
1. **Anime.js** - Smooth micro-interactions and form animations
2. **ECharts.js** - Interactive data visualizations for dashboards
3. **Splide.js** - Testimonial carousels and image galleries
4. **Typed.js** - Dynamic text effects for hero sections
5. **p5.js** - Background particle effects and visual elements
6. **Pixi.js** - Advanced visual effects for hero backgrounds
7. **Matter.js** - Physics-based animations for interactive elements

### Effect Implementation
- **Hero Background**: Subtle particle system with floating geometric shapes
- **Text Effects**: Typewriter animation for main headlines
- **Form Interactions**: Smooth field focus animations and validation feedback
- **Data Visualization**: Animated charts showing claim statistics and performance
- **Scroll Animations**: Gentle fade-in effects for content sections
- **Hover States**: 3D tilt effects on cards and buttons
- **Loading States**: Skeleton screens and progress indicators

### Header Effect
- Clean navigation with subtle backdrop blur
- Smooth scroll-triggered opacity changes
- Mobile-responsive hamburger menu with slide animations

### Styling Approach
- Consistent 8px grid system for spacing
- CSS custom properties for theme consistency
- Responsive design with mobile-first approach
- Accessibility-compliant color contrasts
- Smooth transitions throughout the interface
